# electric-pole-detection-system
This project is an intelligent safety system that detects abnormal current in electric poles using IoT sensors and Deep Learning techniques.It provides real-time alerts through LED, voice, WhatsApp, and Email to prevent electrical hazards.

Due to GitHub size limitations, the dataset is not included in this repository.
